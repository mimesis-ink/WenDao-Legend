## 推荐AI工具参数设置

### Midjourney
添加以下参数到提示词后：
```
--ar 2:3 --q 2 --style expressive --v 6
```

### DALL-E 3
- Quality: HD
- Style: Natural（写实风格）或 Vivid（鲜艳风格）
- Size: 1024x1792（竖版）

### Stable Diffusion
在提示词基础上添加：

**正面提示词（Positive）：**
```
masterpiece, best quality, highly detailed, Chinese mythology, ancient books, golden text, epic composition, cinematic lighting, 8k wallpaper
```

**负面提示词（Negative）：**
```
low quality, blurry, distorted, watermark, signature, text, modern cars, anachronistic, ugly, bad anatomy
```

## 使用建议

1. **批量生成**：每个提示词建议生成10-20张，挑选最佳效果
2. **微调**：根据初次生成结果，适当调整提示词中的关键词
3. **组合使用**：可以将不同方案的元素组合，创造独特效果
4. **后期处理**：AI生成后建议请设计师进行文字排版和细节优化
5. **分辨率**：如用于印刷，确保最终图片至少300dpi

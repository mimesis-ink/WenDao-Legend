# 第067章 丁字班的追随

走在下山的官道上。

“老大！等等俺！”

身后传来了一阵地动山摇的脚步声。

回头一看。

铁柱背着一口巨大的黑锅（真的是锅），手里提着一把锄头，狂奔而来。

在他后面，是一边跑一边算卦的神棍神棍，和背着一把生锈铁剑、气喘吁吁的赵长风。

就连那个胖子朱富贵也追了上来，背上挂满了金银细软，像个移动的聚宝盆。

“你们……”萧千卷愣住了。

“俺退学了。”铁柱憨厚地挠了挠头，“书院的饭太难吃了，还是跟着老大混有肉吃。”

“我也退了。”神棍摇了摇头里的铜钱，“卦象显示，跟着你有肉吃……啊不，是大吉。”

赵长风扶着膝盖，眼神坚定：“我要去真正的战场。书院的沙盘太小了。”

朱富贵嘿嘿一笑：“我想把生意做到长安去。老大你是潜力股，我得投资。”

看着这几个奇葩，萧千卷心里一阵暖流。

“你们知道我去长安是干什么吗？那可是造……咳咳，那可是很危险的事。”

“只要管饭就行。”铁柱说。

“生死有命，富贵在天。”神棍说。

“虽千万人，吾往矣。”赵长风说。

“我带钱了。”朱富贵拍了拍胸口。

萧千卷笑了。

“好。那就一起去。”

“让这天下看看，我们丁字班也不是好惹的！”

于是。

一支由儒（萧千卷）、墨（墨染）、农（铁柱）、阴阳（神棍）、兵（赵长风）、商（朱富贵/后勤）组成的奇怪队伍，正式成团。

这也是未来那个让整个修真界闻风丧胆的“百家天团”的雏形。

（第六十七章 完）

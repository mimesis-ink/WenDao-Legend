# 第十六章 破碗里的道理

江水滔滔，孤舟一叶。

清风县的轮廓早已消失在远方的迷雾中，两岸青山连绵起伏，如同一幅水墨画卷在眼前缓缓展开。江面宽阔，波光粼粼，偶尔有水鸟掠过水面，激起一圈圈涟漪。微风拂过，带来江水特有的腥气，混杂着远处芦苇丛的清香。

天空湛蓝如洗，几朵白云悠悠飘过。阳光洒在江面上，将水波照得金光闪闪，晃得人眼睛发疼。远处的山峰在薄雾中若隐若现，给人一种飘渺虚幻的感觉。

"哗啦……哗啦……"
艄公是个哑巴老头，黝黑的脸上刻满了岁月的痕迹，皱纹如同沟壑。他穿着一身灰色的粗布衣服，袖子挽到肘部，露出满是老茧的臂膀。只是机械地摇着橹，动作熟练而有节奏，橹在水中划动，发出有规律的水声。

船舱内，气氛有些诡异。

老叫花子毫无形象地躺在唯一的软塌上，那软塌虽然旧了些，但料子还算上乘，是这艘船上最好的东西。他翘着二郎腿，一只脚还在空中晃来晃去，脚上的破草鞋已经磨得只剩半只，脏兮兮的脚趾头露在外面。

手里拿着一只烧鸡狂啃，油脂顺着嘴角流下，滴在衣襟上。那只烧鸡外焦里嫩，散发着诱人的香味，不知道是从哪儿弄来的。老叫花子吃得满嘴流油，啃得咔咔作响，时不时还砸吧着嘴，发出满足的声音。

墨染坐在角落里，正在给机械臂做保养。她小心翼翼地打开臂甲的盖板，露出里面精密的齿轮和传动装置。用一把小刷子清理着齿轮缝隙中的灰尘和污垢，然后滴上几滴特制的润滑油。齿轮在她的调试下缓缓转动，发出轻微的咔嚓声。

萧千卷则蹲在老叫花子面前，像个好学的小学生一样，眼巴巴地看着那只烤鸡。他的口水都快流下来了，从昨天到现在，除了啃了几口干粮，他就没吃过像样的东西。

"老大……啊不，前辈。"
萧千卷殷勤地递上一壶酒，那是从船家那里买来的黄酒，不算好酒，但也不算差。装在一个青瓷酒壶里，壶身上画着简单的山水图案。

"您刚才那一招'空间挪移'，能不能教教我？"

那可是逃跑神技啊！学会了这个，以后惹了事直接瞬移，谁也抓不到。想跑就跑，想走就走，岂不美滋滋？

老叫花子斜了他一眼，那眼神中带着一丝玩味，接过酒壶灌了一大口。酒水顺着喉咙流下，他舒服地打了个酒嗝，酒气扑鼻。

"教你？你知道那是什么境界吗？"
他伸出一根油腻的手指，在空中比划了一个圆，手指上还粘着鸡肉的碎屑。

"儒家九品，名为【开窍】。也就是常说的读书读通了，能感应到天地文气，算是入门了。大多数读书人一辈子都困在这个境界，只能算是粗通文墨。"

"八品，名为【修身】。在这个阶段，文气滋养身体，身体素质会大幅提升，百病不生，延年益寿。此时身体才能承载更多的文气，就像水桶的容量变大了，能装更多水。"

"七品，名为【立言】。到了这个境界，就能做到言出法随，说的话带着天地之力。就像你刚才看到的那样，金口玉言，说啥是啥。当然，也不是什么话都能实现，得符合天理人情才行。"

"至于那一招'画圆为界'……"老叫花子嘿嘿一笑，又啃了一口鸡腿，"那是'亚圣'级别的手段，掌握空间法则，弹指间可破碎虚空。你小子离那还差着十万八千里呢，连九品都不是，就想学亚圣的招数？做梦呢！"

萧千卷听得目瞪口呆，嘴巴张得能塞下一个鸡蛋。

"合着我现在连九品都不是？"他的声音都变了调，带着一丝绝望。

"你？"老叫花子上下打量了他一番，眼中闪过一丝怜悯，"你顶多算个……半步九品？或者说，是个满肚子墨水却倒不出来的……嗯，怎么说呢，就像茅坑里的石头，又臭又硬，憋得慌。"

"噗——"角落里的墨染没忍住，笑出了声。她连忙捂住嘴，但肩膀还在微微颤抖。

萧千卷一脸黑线，额头上青筋直跳。这什么比喻啊！虽然很形象，但也太侮辱人了吧！

"那我要怎么才能'倒出来'？"他咬着牙问道，拳头都攥紧了。

老叫花子没有直接回答，而是放下鸡腿，把他那个缺了口的破瓷碗放在桌上。碗里还有些酒渍，在阳光下泛着微光。

"阿巴，吐水。"

正在萧千卷肩膀上睡觉的汤圆被吵醒，半睁着眼睛，很不情愿地吐了一口口水在碗里。那是黑色的墨水，浓稠得像胶水，在碗中缓缓流淌。

"你看这碗里的水。"老叫花子指着破碗，语气变得认真起来，"水就是文气，碗就是你的身体。现在你的情况是，水太多，碗太小，而且这碗……"

他敲了敲碗沿，发出清脆的声响，"还是漏的。"

"漏的？"萧千卷大惊，下意识地摸了摸自己的身体，"我……我肾亏？"

"滚犊子！"老叫花子一巴掌拍在他脑门上，打得他脑袋一歪，"是你的'窍'没开！文气在你体内是一潭死水，没有流动，就没有力量。就像水塘里的水，时间长了就会发臭。儒家的修行，讲究的是'胸中有沟壑，笔下起风雷'。你得把这些死水，变成活水，让它流动起来，形成循环。"

"怎么变？"萧千卷揉着脑袋问道，这一巴掌打得还真疼。

"读万卷书，行万里路。"
老叫花子指了指窗外的江面，语气变得悠远起来。

"看见这江水了吗？它无论遇到高山还是深谷，都在流动，都在变化。遇到石头就绕过去，遇到悬崖就跳下去，遇到平原就铺开来。它从不停止，从不固执，这就是'道'。"

"你什么时候能把你脑子里的那些死道理，像这江水一样用活了，灵活变通，融会贯通，你就真正入门了。记住，死读书不如不读书，读死书只会把自己读死。"

萧千卷若有所思地看着窗外，眼神渐渐变得深邃。

江水在阳光下波光粼粼，不断流淌，绕过岩石，越过浅滩，永不停歇。它时而平缓，时而湍急，但始终向前，奔向大海。

活用……死道理？

就像之前用《大稷律》砸人，或者用"画地为牢"喂虫子？不拘泥于形式，只求达到目的？

"不对劲。"
墨染突然站了起来，打断了他的沉思。她走到船头，眉头紧锁地看着前方，手已经按在了短刃上。

"起雾了。"

并不是普通的江雾。

前方的雾气渐渐浓厚起来，将整个江面笼罩。这雾气泛着淡淡的粉红色，在阳光下显得格外诡异。空气中弥漫着一股甜腻的香味，像是脂粉的味道，又像是某种花香，闻多了让人头晕目眩。

原本平静的江面，突然变得死寂一片，连水鸟的叫声都听不见了。就连那单调的橹声都停止了，整个世界仿佛被按下了静音键。

"哑巴艄公不见了。"墨染冷冷说道，声音中带着警惕，左眼的机械瞳孔开始快速旋转，扫描着周围。

萧千卷一惊，连忙跑出去一看。

果然，船尾空空如也，那根长长的船橹还在水里晃荡，发出吱呀的声响，但摇船的人却像是凭空蒸发了一样。甚至连衣服都没留下，就这么消失得无影无踪。

水面上泛起阵阵涟漪，仿佛有什么东西刚刚潜入水中。

"这是……桃花瘴？"
老叫花子嗅了嗅鼻子，却丝毫没有紧张的意思。他甚至还打了个哈欠，伸了个懒腰，然后翻了个身，把脸朝向里侧，拉过一件破旧的袍子盖在身上。

"哎呀，困了困了。人老了就是觉多，经不起折腾了。前面这点小麻烦，你们年轻人自己解决吧。老大我要睡了。"

"喂！老大！别睡啊！"
萧千卷慌了，连忙摇他的肩膀，"这明显是遇到黑店……啊不，黑船了啊！有妖怪啊！你不能见死不救啊！"

"年轻人，要有历练。"
老叫花子的呼噜声瞬间响起，打得震天响，"要是连这点小风浪都过不去，还去什么长安？不如回家种红薯。反正死不了，放心好了。"

他说完就真的睡着了，鼾声如雷，震得船板都在颤动。

萧千卷气得牙痒痒。这老头，关键时刻掉链子！说好的保护呢？说好的有吃有喝呢？现在遇到危险了，就让他自己应对？

就在这时。

咚！
一声闷响，整艘船剧烈摇晃起来。

乌篷船像是撞到了什么坚硬的东西，猛地停了下来。船板发出吱吱呀呀的声响，仿佛随时会散架。船舱内的东西都被震得跳了起来，萧千卷一个没站稳，差点摔倒。

前方的迷雾中，慢慢浮现出一艘巨大的楼船阴影。那楼船高耸入云，少说有五六层楼高，在雾中若隐若现，如同一座水上宫殿。

楼船挂着大红灯笼，一排排整齐地挂在船舷两侧，少说有上百个。但灯笼里透出的光却是惨绿色的，如同鬼火般闪烁，在粉红色的雾气中显得格外诡异。

船上隐隐传来靡靡之音，是女子的娇笑声和丝竹之声，缠绵悱恻，令人心神荡漾。空气中的脂粉味更浓了，浓到几乎让人窒息。

"红粉骷髅舟。"
墨染拔出短刃，横在胸前，挡在萧千卷身前。她的脸色变得凝重无比，额头上渗出了细密的汗珠。

"小心，这是'合欢宗'外围势力的水匪，专门劫掠过往书生和精壮男子，抓走后做……"

她顿了一下，似乎有些难以启齿，咬着牙说道："做人肉鼎炉。采补修炼，榨干精气，最后扔到江里喂鱼。"

萧千卷顿时感觉下半身一凉，浑身的鸡皮疙瘩都起来了。他下意识地双手护胸，身体微微后退。

"劫财就算了，还劫色？！"他的声音都在颤抖，"这也太丧心病狂了吧！"

巨大的楼船缓缓靠近，船头雕刻着一个妖艳女子的塑像，双眼是两颗血红色的宝石，在绿光中闪烁着诡异的光芒。塑像的嘴角勾起一个诡异的笑容，仿佛在嘲笑着即将成为猎物的人。

雾气越来越浓，几乎伸手不见五指。甜腻的香气也越来越浓，萧千卷已经开始感觉头晕，眼前的景物都开始模糊起来。

墨染咬破舌尖，用疼痛保持清醒。她的机械臂开始蓄能，发出低沉的轰鸣声。

"准备战斗。"她冷冷地说道，"能杀多少杀多少，实在不行……"

她看了一眼还在呼呼大睡的老叫花子，眼中闪过一丝无奈和愤怒。

这位"高人"，到底靠不靠谱啊！

（第十六章·完）

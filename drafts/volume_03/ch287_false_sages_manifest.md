# 第二百八十七章 伪圣显灵

虽然萧千卷不敢用了，但有的是人敢用，甚至争着用。
在逃亡路过一个村庄时，他们看到了一场诡异的祭祀。

村里的打谷场上，立着一尊黑色的雕像。
那雕像长着孔子的脸，却有着章鱼的触手和獠牙。
一群穿着破烂儒衫的人，正在对着雕像疯狂磕头，嘴里念叨着刚颁布的《新修大典》里的句子。

“圣人显灵了！赐予我们力量吧！”
随着他们的狂热祈祷，那雕像的眼睛亮起了红光。
黑色的触手从地下伸出，刺入那些信徒的后脑。
信徒们不仅没有痛苦，反而露出了极度愉悦的表情。他们的身体开始膨胀，肌肉撕裂衣衫，变成了力大无穷但毫无理智的怪物。

“这也配叫圣人？”墨染看得想吐，“这明明是把人变成丧尸！”

“这就是‘伪圣’。”萧千卷压低帽檐，挡住眼中的杀意，“在这个时代，只要能给人力量，哪怕是把人变成鬼，也有人跪在地上喊爹。”

“走吧，别看了。”萧千卷拉着墨染，“我们救不了他们。他们已经把灵魂卖了。”

（第二百八十七章·完）
